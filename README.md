# 🚲 Site Bike

> Projeto desenvolvido durante o curso **Fábrica de Software**, ministrado pela professora **Enilda**.

---

## 📚 Sobre o Projeto

O **Site Bike** é um site temático sobre bicicletas, criado como parte das aulas práticas do curso Fábrica de Software. O projeto foi construído do zero, acompanhando o aprendizado progressivo de **HTML** e **CSS**, desde os conceitos mais básicos até a estruturação de páginas web completas.

## 💡 O que foi Aprendido

Ao longo do desenvolvimento deste projeto, foram praticados os seguintes conceitos:

- Estrutura básica de um documento HTML
- Tags semânticas (`header`, `main`, `footer`, `section`, etc.)
- Formatação de texto (títulos, parágrafos, listas)
- Inserção de imagens e links
- Estilização com CSS (cores, fontes, margens, padding)
- Modelo de caixas (Box Model)
- Layout e posicionamento de elementos
